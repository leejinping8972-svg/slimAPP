# 规则 — 每日记录与评分

> 产品：luckdate · 版本 V2.2 · 2026-07-24  
> 范围：记什么、怎么算分、怎么展示。不含技术实现。  
> 对应主文档：`PRD-luckdate.md`  
> 对应界面：Journey 分数环与五大维度（**不含冥想 Mindfulness**）

---

## 1. 原则

- **行为优先：** 奖励记录与靠近目标的进度，而不是奖励体重下降。  
- **不惩罚情绪：** 睡眠状态差不额外「羞辱扣分」；用温和分值 + Sunny 关怀表达。  
- **低焦虑：** 稳定性用「缺记逐步掉分、再记逐步加回」，文案避免「断签失败」。  
- **可解释：** 饮水看进度，睡眠看入睡/起床/状态，习惯看连续缺记或恢复天数。  

分数用于「我在坚持」的反馈，**不用于诊断健康或承诺减重效果。**  
**体重数值升/降不改变当日分数**，只影响「是否已记录体重」。

## 2. 每日可记录项

| 记录项 | 评分方式 | 备注 |
|--------|----------|------|
| 产品使用 | 有记录 **100** / 无 **0** | 不区分吃全/吃一半 |
| 饮水 | **按相对目标进度 0～100**（见 5.2） | 默认目标约 2000ml，可改 |
| 体重 | 有记录 **100** / 无 **0** | 数值本身不加分不扣分 |
| 状态反馈 | 有记录 **100** / 无 **0** | 负面心情仍 100，触发关怀 |
| 睡眠 | **按入睡、起床、睡眠状态 0～100**（见 5.5） | 需尽量记两端时间 + 状态 |
| 饮食（可选） | 计入 Nutrition：有记 100 / 无 0 | — |
| 运动（可选） | 计入 Exercise：有记 100 / 无 0 | — |

## 3. 对外指标一览（Journey 五大指标）

全部为 **0～100 分数**（不用 % 作主数字）。

| 指标 | 范围 | 业务含义 | 对用户怎么说 |
|------|------|----------|--------------|
| **每日生命力分** | 0～100 | 当日加权综合（第 4 节） | 今天的生命力状态 |
| **Nutrition** | 0～100 | 有产品或饮食记录 → 100；否则 0 | 营养记录 |
| **Exercise** | 0～100 | 有运动记录 → 100；否则 0 | 运动记录 |
| **Sleep** | 0～100 | 入睡 + 起床 + 睡眠状态综合（5.5） | 睡眠恢复感 |
| **Hydration** | 0～100 | 饮水相对目标进度（5.2） | 饮水进度 |
| **Habits** | 0～100 | 连续缺记扣分 / 再记加回（5.6） | 习惯稳定性 |

## 4. 每日生命力分（加权口径）

$$
\text{Daily Vitality} = \mathrm{round}\big(
  S_{产品}\times0.25 +
  S_{体重}\times0.25 +
  S_{饮水}\times0.15 +
  S_{状态}\times0.15 +
  S_{睡眠}\times0.10 +
  S_{稳定}\times0.10
\big)
$$

| 维度 | 权重 | 子分范围 |
|------|------|----------|
| 产品仪式 | **25** | 0 或 100 |
| 体重记录 | **25** | 0 或 100 |
| 饮水 | **15** | **0～100**（进度分） |
| 状态反馈 | **15** | 0 或 100 |
| 睡眠 | **10** | **0～100**（细化分） |
| 温和稳定性 | **10** | **0～100**（5.6） |

**示例：** 产品/体重/状态各 100，饮水 80（未满目标），睡眠 90，稳定 100 →  
$25+25+12+15+9+10 = 96$。

## 5. 各子项判定（具体分值）

### 5.1 产品仪式分 $S_{产品}$

| 情况 | 分值 |
|------|------|
| 今日已记录产品使用 | **100** |
| 无记录 | **0** |

### 5.2 饮水分 $S_{饮水}$（进度分）

设：今日饮水量为 $W$，饮水目标为 $T$（默认 2000ml，用户可改；$T>0$）。

$$
S_{饮水} =
\begin{cases}
0 & W = 0\text{ 或未记录} \\
100 & W \ge T\text{（达标）} \\
\mathrm{round}\big(100 \times W / T\big) & 0 < W < T\text{（越接近目标越高）}
\end{cases}
$$

| 示例（$T=2000$） | $W$ | $S_{饮水}$ |
|------------------|-----|------------|
| 未喝 | 0 | **0** |
| 喝了一半 | 1000 | **50** |
| 接近目标 | 1800 | **90** |
| 达标或超过 | ≥2000 | **100**（超过不额外加分） |

目标过低/过高建议二次确认，避免分数失真。

### 5.3 体重记录分 $S_{体重}$

| 情况 | 分值 |
|------|------|
| 今日已记录体重 | **100** |
| 未记录 | **0** |

升降不改分。

### 5.4 状态反馈分 $S_{状态}$

| 情况 | 分值 |
|------|------|
| 有心情/精力等任一记录 | **100** |
| 无 | **0** |

### 5.5 睡眠分 $S_{睡眠}$（入睡 / 起床 / 状态）

用户尽量记录：

- **入睡时间**（昨晚/今日入睡）  
- **起床时间**  
- **睡眠状态**：good / okay / poor（或等价文案）

由入睡与起床得到睡眠时长 $H$（小时）。若只缺一端时间，无法算准确时长，按「不完整」档计分。

#### 5.5.1 基础分（时长 + 完整度）

| 记录完整度 | 时长 $H$ | 基础分 $B$ |
|------------|----------|------------|
| 无任何睡眠信息 | — | **0** |
| 仅有状态，无入睡/起床 | — | **40** |
| 仅有入睡或仅有起床（缺一端） | — | **50** |
| 入睡 + 起床齐全 | $H < 6$ | **60** |
| 入睡 + 起床齐全 | $6 \le H < 7$ | **80** |
| 入睡 + 起床齐全 | $H \ge 7$ | **90** |

#### 5.5.2 状态微调

在基础分 $B>0$ 上：

| 睡眠状态 | 调整 |
|----------|------|
| good | **+10**（上限 100） |
| okay / 未填状态但已有两端时间 | **+0** |
| poor | **−10**（下限不低于 40；若仅有状态无时间则保持 40） |

$$
S_{睡眠} = \mathrm{clamp}\big(B + \Delta_{状态},\ 0,\ 100\big)
$$

| 示例 | 分值 |
|------|------|
| 23:00 睡，7:00 起（8h），状态 good | $90+10$ → **100** |
| 23:00 睡，6:00 起（7h），状态 okay | **90** |
| 1:00 睡，6:30 起（5.5h），状态 poor | $60-10$ → **50** |
| 只选了「睡得一般」 | **40** |
| 完全没记 | **0** |

差睡眠：Sunny 建议减轻当日任务，**文案不羞辱**；分数如实反映完整度与恢复感。

---

### 5.6 温和稳定性分 $S_{稳定}$（Habits）— 缺记扣分 / 再记加回

#### 5.6.1 何为「当日有记录」

**任意一项**有记录即算当日有记录（完成门槛很低）：

产品 **或** 饮水 **或** 体重 **或** 状态 **或** 睡眠 **或** 饮食 **或** 运动。

#### 5.6.2 连续缺记天数 $G$（gap）

从 **今天** 往回数：连续「完全没有任何记录」的天数。

- 今天有任意记录 → $G = 0$  
- 今天无记录、昨天有 → $G = 1$  
- …  
- 连续 7 天（含今天）都无记录 → $G = 7$

#### 5.6.3 主规则：有记录从满分起，缺记逐步扣到 0

**常态（未处于「归零后恢复期」）：**

| 连续缺记天数 $G$ | $S_{稳定}$ | 含义 |
|------------------|------------|------|
| **0**（今天有任意记录） | **100** | 第一天/当天完成记录即满分 |
| 1 | **86** | 缺 1 天，开始扣分 |
| 2 | **71** | |
| 3 | **57** | |
| 4 | **43** | |
| 5 | **29** | |
| 6 | **14** | 只剩「很远的一天曾有过记录」感，接近最低 |
| **≥ 7** | **0** | 连续 7 天无任何记录 |

等价公式（常态）：

$$
S_{稳定} = \mathrm{round}\left(100 \times \frac{7 - \min(G,\ 7)}{7}\right)
$$

#### 5.6.4 归零后的恢复期（逐步加回）

当曾经达到 $G \ge 7$（分数为 0）之后，进入 **恢复期**，直到连续有记录天数攒满 7 天。

设 $R$ = 归零之后，从重新开始记录起算的 **连续「当日有记录」天数**（当天有任意记录才计入）。

| 恢复连续天数 $R$ | $S_{稳定}$ |
|------------------|------------|
| 0（仍无记录） | **0** |
| 1 | **14** |
| 2 | **29** |
| 3 | **43** |
| 4 | **57** |
| 5 | **71** |
| 6 | **86** |
| ≥ 7 | **100**（退出恢复期，回到 5.6.3 常态） |

等价公式（恢复期）：

$$
S_{稳定} = \mathrm{round}\left(100 \times \frac{\min(R,\ 7)}{7}\right)
$$

#### 5.6.5 行为小结

```
今天有任意记录且不在恢复期 → 100
缺记 1～6 天 → 86…14 逐步降
连续 7 天无记录 → 0，进入恢复期
恢复期每天有任意记录 → 14→29→…→100，满 7 天退出恢复期
```

文案：用「今天记一件就能回到满分节奏」「最近几天空了，记一下就好」，避免「断签」。

**示例：**

| 情况 | $S_{稳定}$ |
|------|------------|
| 今天记了饮水 | **100** |
| 今天没记，昨天记过 | **86** |
| 连续 3 天没记 | **57** |
| 连续 7 天没记 | **0** |
| 归零后第 1 天又记了体重 | **14** |
| 归零后连续第 7 天都有记 | **100**（退出恢复期） |

## 6. Journey 五大维度分

| Journey 维度 | 分值规则 |
|--------------|----------|
| **Nutrition** | 有产品或饮食记录 → **100**；否则 **0** |
| **Exercise** | 有运动记录 → **100**；否则 **0** |
| **Sleep** | 同 $S_{睡眠}$（5.5）→ **0～100** |
| **Hydration** | 同 $S_{饮水}$（5.2）→ **0～100** |
| **Habits** | 同 $S_{稳定}$（5.6）→ **0～100** |

**不展示**冥想环。生命力环用第 4 节加权。

---

## 7. 展示文案方向（避免伤害）

| 生命力分区间 | 表达方向 | 避免 |
|--------------|----------|------|
| 90～100 | 节奏很好 | 「完美」「马上会瘦」 |
| 75～89 | 很稳定 | 「还差一点才合格」 |
| 50～74 | 慢慢接上 | 「低分」「失败」 |
| 1～49 | 一小步也算 | 「拖后腿」 |
| 0 | 随时可以重新开始 | 「断签」 |

睡眠 poor：可说「今天把任务放轻一点」，不说「你睡太差了所以分低」。

## 9. Journey 每日建议与评分

- 建议可展示饮水目标与睡眠窗口；**饮水达标影响饮水分**；睡眠建议不替代 5.5 计分。  
- 稳定性：一天只需任意一项记录即可维持/恢复分数。  

---

## 10. 验收要点

- [ ] 饮水：达标 100；未达标按 $W/T$；未记录 0；超标不额外加分  
- [ ] 睡眠：综合入睡、起床、状态；无记录 0；可手算示例表  
- [ ] 产品/体重/状态仍为 100/0  
- [ ] 稳定：当天任意记录 → 100；缺记按 7 档扣到 0；归零后按 $R$ 逐步加回  
- [ ] Journey 五环无冥想；Sleep/Hydration/Habits 可为非 0/100 中间分  
- [ ] 文案无断签羞辱
